# SourceSnap - Source Navigator

SourceSnap is a powerful developer tool that allows you to instantly jump from a browser element to its source code in your editor.

## Documentation & Repository

- **Main Repository**: [SourceSnap on GitHub](https://github.com/ChinmayShringi/sourcesnap)
- **Extension Link**: [SourceSnap Chrome Extension](https://github.com/ChinmayShringi/sourcesnap)

## Quick Start

1. **Install the Extension**: Download and install the SourceSnap extension from the link above.
2. **Open your project**: Run your React application locally with source maps enabled.
3. **Snap to Source**: Press `Alt + Click` on any element in your browser to jump directly to the code in your preferred editor.

## Development

If you want to contribute to the Source Navigator landing page codebase:

```sh
# Clone the repository
git clone https://github.com/ChinmayShringi/sourcesnap.git

# Install dependencies
npm i

# Start development server
npm run dev
```
